# Instagram scraper

### Prerequisites
 - nodejs (tested on ver 10)
 - yarn or npm
 
### Installation
```bash
git clone git@github.com:oleksii-dmytrenko/instagram-scraper.git
cd instagram-scraper
yarn install
```

### Usage
```bash
yarn start
```
